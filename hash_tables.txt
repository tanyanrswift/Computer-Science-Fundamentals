1. What is a hash table?
  A hash table is a collection of values much like an array that has key and value pairs.

2. What is hashing?
  Hashing is the method of converting strings to numerical indexes to be used in a hash table.

3. How does a hash table store data?
  A hash table stores data in key and value pairs, with the value holding the data and the key linking to the index of the value.

4. How are hash tables and objects different?
  Objects bundle different types of data together, whereas hash tables store collections of similar data together. In other words, each object stores data
  that is similar but each object is different, and a hash table would store data that has some sort of identifier.
